# Janus Additions

## Pre-reqs

- Your vim of choice
- [carlhuda/janus](https://github.com/carlhuda/janus)

## Install

    git clone git@github.com:igreulich/.janus ~/.janus
    git submodule init
    git submodule update

## Add

    git submodule add [git repo address]

## Update

    git submodule foreach git pull origin master
